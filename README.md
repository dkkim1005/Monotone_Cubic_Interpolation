Monotone Cubic Interpolation
============================================================

Implementation for **monotone cubic interpolation** with the Hermite function through python3 and C++

Environment Setup
--------------

+ python3
  + numpy
  + matplotlib (optional)
+ C++
  + Standard c++11 or higher version is required.
